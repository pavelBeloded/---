CSS (Cascading Style Sheets) используется для стилизации веб-страниц Основные категории свойств CSS включают: Свойства текста (например, font-size, color, text-align) Свойства фона (например, background-color, background-image) Свойства размера и позиционирования (например, width, height, position) Свойства границ и отступов (например, border, margin, padding) Свойства макета (например, display, flex, grid) Каждое свойство имеет свой синтаксис и допустимые значения
